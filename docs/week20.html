Implementation of features from the client

Posted on February 21, 2022

Following a client meeting, he seemed overall happy with everything we had done so far but made somesuggestions for useful features that could be implemented. He wanted to see some features to help the user understand what some of the features do. In addition a video to show how to use the website.

We implemented a help-tip feature which is displayed as a question mark on the required sections. When hovered, it reveals a helpful tip for the user that explains the graphs and the helps users understand what ngrams are.
