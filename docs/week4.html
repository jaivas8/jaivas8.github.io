Week 4

Gathering user requirements

Posted on October 31, 2021

Given the goal of our project - to generate Ngrams from word clouds, we began brainstorming on how to approach the task.

We first created a MoSCoW to lay out the possible additions to the current web app.

From the MoSCoW, we summarised some features that we could implement for the web app:

  • Analyse the word cloud through gathering the frequent words, or search for keywords from a dataset. If the keywords from the dataset are present in the data provided by a patient, we send the details of the patient to the backend system.
  • Determine whether a certain key phrase is context appropriate.
  • Improve the UI of the current website
