Week 3

1st meeting with client and understanding of the project

Posted on October 24, 2021

We met our client via Zoom and were introduced to the background and initiatives of our project.

Understanding our project

After our meeting with the client, we were provided with a Trello board that stored information about other previously completed projects that were related to ours.

What we know about our project so far

The Trello board was loaded with plenty of information about the related projects, but not so much about our own project.

Background

  • Our project is about adding functionality to an existing web application. The web app mainly gathers text and voice recordings from patients in the UK and generates a *word cloud* for further analysis.
  • We began researching about word clouds and natural language processing to get a better gist about the context of our project. These resources can be found in our research page.

    • Ultimate goal of the web application

    To allow users to analyse the text and voice recordings through generating word clouds and Ngrams.

    State of the current web application

    The current web application only supports the generation of a word cloud and has a lot of bug issues

    Our goal is to add a feature to allow for the creation of meaningul ngrams

    Who are the users?

    NHS researcher will use the web app to upload and analyse the text and voice recordings
